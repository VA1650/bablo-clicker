# 💸 Bablo Clicker

Простой прототип кликера (idle game), реализованный на чистом JavaScript.

## 🎮 Функционал

- Система дохода от кликов
- Пассивный доход (CPS)
- Система престижа
- Банк с процентной ставкой
- Динамические цены на улучшения
- Случайные события
- Сохранение прогресса (localStorage)

## 🧠 Технологический стек

- HTML
- CSS
- JavaScript (без фреймворков)

## 🚀 Как запустить

Просто откройте `index.html` в браузере.

## 📌 Статус

Прототип / активная разработка idle-игры.
