# 💸 Bablo Clicker

A simple idle clicker / tycoon prototype built in vanilla JavaScript.

## 🎮 Features

- Clicking income system
- Passive income (CPS)
- Prestige system
- Bank with interest
- Dynamic upgrade prices
- Random events
- Offline progress saving (localStorage)

## 🧠 Tech

- HTML
- CSS
- JavaScript (no frameworks)

## 🚀 How to run

Just open `index.html` in browser.

## 📌 Status

Prototype / work in progress idle game.
